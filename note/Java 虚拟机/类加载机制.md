## 虚拟机类加载机制



### 类加载时机

Java虚拟机规范并没有规定什么时候需要进行类的加载阶段，但是**对一个类进行主动引用时会触发类的初始化**。

5 种场景：

- 使用 new 关键字实例化对象、读取或设置类的静态字段 (被 final 修饰，已在编译期把结果放入常量池的静态字段除外)、以及调用一个类的静态方法的时候。
- 使用 java.lang.reflect 包的方法对类进行反射调用的时候。
